# Threat Hunting & Forensic Analysis Toolkit

## Project Description
This project is a Python-based Threat Hunting and Forensic Analysis Toolkit designed to detect suspicious activities from system logs. The system performs IOC detection, timeline reconstruction, process scanning, and generates structured forensic reports.

## Features
- Log file parsing
- Indicator of Compromise (IOC) detection
- Timeline reconstruction
- Suspicious process scanning
- Automated forensic report generation
- Graphical User Interface (GUI)

## Technologies Used
- Python 3
- Tkinter (GUI)
- psutil
- CSV module

## How to Run
1. Install Python
2. Install required package:
   pip install psutil
3. Place iocs.txt and sample_logs.log in same directory
4. Run:
   python main.py
